-------JSON.parse()
JSON.parse() is used to convert JSON string into JS object:


const jsonString = '{"name": "John", "age": 30}';
const JSObject = JSON.parse(jsonString);
console.log(jsonObject.name); // Output: John


---------------JSON.stringify()
This method is an opposite of JSON.parse() This is used to convert JS object into JSON string

const jsObject = {name: "John", age: 30};
const jsonString = JSON.stringify(jsonObject);
console.log(jsonString); // Output: {"name":"John","age":30}

-----------------.json()
its not a built in methd it is provided by a library such as fetch API

This is used to extract the json body when an API is called:

fetch('https://example.com/api/data')
  .then(response => response.json())
  .then(data => {
    console.log(data);
  });
